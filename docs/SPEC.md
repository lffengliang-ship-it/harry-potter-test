# 哈利波特性格测试 - 设计规格文档

**项目名称**: 哈利波特学院性格测试
**类型**: H5 互动测试页面
**创建日期**: 2026-04-29

---

## 1. 项目概述

### 核心功能
在哈利波特世界观下，通过 8-10 道情境选择题，让用户知道自己：
1. 属于霍格沃茨哪个学院
2. 最像该学院的哪个角色
3. 角色的特点、擅长魔法
4. 角色在魔法世界经历过的大事件

### 目标用户
- 哈利波特粉丝
- 喜欢趣味心理测试的用户
- 16-35 岁年轻群体

---

## 2. 技术架构

### 技术选型
- **纯 HTML/CSS/JS**，单文件部署
- **Canvas API** 绘制雷达图（如需要）
- **GitHub Pages** 部署

### 项目结构
```
harry-potter-test/
├── index.html          # 主页面（含全部逻辑）
├── config.js           # 角色图片等配置（可替换）
├── assets/             # 静态资源（角色图片等）
│   └── characters/     # 角色剧照文件夹
└── README.md          # 项目说明
```

### 图片配置化设计
- `config.js` 中定义角色 → 图片路径的映射
- 默认提供占位图（CSS 生成，无版权风险）
- 用户只需在 `assets/characters/` 放入图片，修改 `config.js` 中的路径即可替换
- 无需修改任何业务逻辑代码

## 3. 测试维度设计

### 四学院维度
| 学院 | 核心特质 | 相关角色 |
|------|----------|----------|
| 格兰芬多 | 勇气、冒险、正义 | 哈利·波特、罗恩·韦斯莱、赫敏·格兰杰 |
| 赫奇帕奇 | 忠诚、勤奋、可靠 | 纽特·斯卡曼德、塞德里克·迪戈里 |
| 拉文克劳 | 智慧、理性、博学 | 卢娜·洛夫古德、秋·张 |
| 斯莱特林 | 野心、谋略、领导力 | 德拉克·马尔福、斯内普教授 |

### 计分方式
每道题有 4 个选项，分别对应不同学院维度：
- A → 格兰芬多
- B → 赫奇帕奇
- C → 拉文克劳
- D → 斯莱特林

## 4. 题目设计（8-10 道）

### 原则
- 情境化题目（不是直接问性格）
- 选项有趣、有画面感
- 每题对应不同生活场景
-门槛低，容易理解

### 题目方向（示例）
1. 面对未知危险的第一反应
2. 处理朋友冲突的方式
3. 发现魔法秘密后的选择
4. 面对考试/挑战的态度
5. 选择伙伴的标准
6. 遇到不公待遇的反应
7. 获得强大力量后的选择
8. 最重要的人生信条

---

## 5. 角色库设计

### 每学院 2-3 个代表性角色

#### 格兰芬多
- **哈利·波特**：勇敢、正义、大难不死
- **罗恩·韦斯莱**：忠诚、幽默、愿为朋友牺牲
- **赫敏·格兰杰**：勇敢、智慧、领导者

#### 赫奇帕奇
- **纽特·斯卡曼德**：善良、执着、保护神奇动物
- **塞德里克·迪戈里**：公平竞争、诚实、勇敢

#### 拉文克劳
- **卢娜·洛夫古德**：独特思考、智慧、富有想象力
- **麦格教授**：理性、严谨、有领导力

#### 斯莱特林
- **斯内普教授**：复杂、牺牲、双重身份
- **德拉克·马尔福**：精致利己、摇摆不定

### 角色数据
每个角色包含：
- 名称
- 学院
- 核心特质（3-4 个关键词）
- 擅长魔法（1-2 个）
- 名场面/大事件（1-2 个，有故事性）

## 6. 结果页设计

### 输出内容
1. **所属学院** + 学院标志色
2. **最像的角色** + **角色剧照**
   - 每个角色配一张代表性剧照/插画
   - 结果页居中展示，上方角色名称 + 下方详细解读
   - 剧照需清晰可辨识，尺寸统一（如 300x400px）
   - **版权提醒**：哈利波特剧照属于华纳版权，建议使用自己设计的插画、AI 生成形象，或公共领域同人作品
   - **配置化设计**：角色图片路径由外部配置文件管理，默认使用占位图，用户只需修改配置即可替换
3. **角色特点分析**（100-200 字）
4. **擅长魔法**
5. **名场面故事**（200-300 字，有传播性）

### 视觉风格
- 霍格沃茨魔法风
- 配色：学院色 + 暗色调
- 字体：衬线体（如有）
- 动态效果：渐入、闪烁

## 7. 部署方式

### GitHub Pages
1. 创建 GitHub 仓库：`harry-potter-test`
2. 推送代码到 master/main 分支
3. Settings → Pages → Select master as source
4. 访问：`https://你的用户名.github.io/harry-potter-test/`
